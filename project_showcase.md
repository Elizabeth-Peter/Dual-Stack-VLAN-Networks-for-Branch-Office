# Secure Dual-Stack Branch Office Network
To design a secure and scalable branch office network that supports both IPv4 and IPv6 while separating departments using VLANs.
<body>
  <h1> Design Explanation</h1>
  
## <p>Each department was assigned its own VLAN to:
- Improve security
- Reduce broadcast traffic
- Allow easier management
Router-on-a-Stick was used to enable communication between VLANs.
